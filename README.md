# Tic Tac Toe Kata (JS + Jasmimne)

## Kata description

### Game Setup

- One board – 3 x 3 cells
- Two players – O & X
- Players take turns

### Game Rules

- Player One starts
- A mark can only be placed on an empty position
- Game ends if …
  - board full
  - only one symbol per
  - column, row or diagonal
